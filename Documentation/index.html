Theme Details

  1. Name: OVIK - A Personal Portfolio Template
  2. Created: 11 - 23 July, 2025
  3. Item Version : v 1.0
  4. Author: CodeOVIK

Key Features

  1. HTML 5
  2. CSS 3
  3. Tailwind css 4.1
  4. Google fonts
  5. Fontawsome
  6. Gsap
  7. Responsive Desing
  8. Odometer animation
  9. Typewriter effect
  10. Slider Effect
  11. 3D image hover

File Structure

Theme                                            <== Root folder of portfolio site
├── assets                                       <== All assets likes images, fonts, icons and other codes here
│   ├── css                                      <== All stylesheets here
│   │   ├── all.min.css                          <== Fontawsme icons
│   │   ├── custom.css                           <== Stylesheet consrol like: brand color and fonts
│   │   ├── odometer-theme-default.min.css       <== Number count with odometer animation
│   │   ├── output.css                           <== main stylesheet
│   │   └── swiper-bundle.min.css                <== Clients review's card slider
│   ├── files                                    <== All files for this template
│   │   └── cv.pdf                               <== CV file
│   ├── fonts                                    <== All fonts here
│   │   └── MontserratAlternates-Regular.ttf     <== Default font
│   ├── images                                   <== All images for this template
│   │   ├── client-1.png                         <== Client image
│   │   ├── favicon.png                          <== Favicon image
│   │   ├── food-delivery-app.png                <== Portfolio: food delivery app development screenshort
│   │   ├── hero.png                             <== Hero section image
│   │   ├── logo-dark.png                        <== Dark for dark mode
│   │   └── logo-light.png                       <== Light for light mode
│   ├── js                                       <== All JavaScript files here
│   │   ├── gsap.min.js                          <== GSAP for animations
│   │   ├── odometer.min.js                      <== Odometer for number count animation
│   │   ├── script.js                            <== Main JavaScript file
│   │   ├── ScrollTrigger.min.js                 <== GSAP ScrollTrigger for scroll animations
│   │   ├── swiper-bundle.min.js                 <== Swiper for client review's card slider
│   │   ├── typed.umd.js                         <== Typing effect
│   │   └── vanilla-tilt.min.js                  <== Vanilla tilt for tilt effect
│   └── webfonts                                 <== Fontawsome webfonts
│       ├── fa-brands-400.ttf                    <== Fontawsome brand icons for modern browsers
│       └── fa-brands-400.woff2                  <== Fontawsome brand icons for older browsers
├── index.html                                   <== Main HTML file
└── thanks.html                                  <== Thanks page when click on contact form submit button
